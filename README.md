# Java Basics for Spring Boot Developers

## 📚 About this project
This repository contains all source code from my Udemy course:

👉 *Java Basics for aspiring Spring Boot developers*

The goal is to build a strong Java foundation before moving to Spring Boot.

- Abschnitt 3 – Methoden
- Abschnitt 4 – Klassen und Objekte
- Abschnitt 5 – Objektorientierte Programmierung
- Abschnitt 6 – Kernkonzepte & Techniken
- Abschnitt 7 – Ausnahmebehandlung
- Abschnitt 8 – Datenstrukturen & Algorithmen
- Abschnitt 9 – Generics
- Abschnitt 10 – Lambda Ausdrücke
- Abschnitt 11 – Streams
- Abschnitt 12 – Multithreading
- Abschnitt 13 – Datenbanken
- Abschnitt 14 – Toolbox Demo
